# EnvSeed Developer Specification
This document defines the normative specification for EnvSeed, including parsing and rendering behavior, security properties, conformance requirements, and the CLI contract. This document is not a user guide.
The key words `MUST`, `MUST NOT`, `SHOULD`, and `MAY` are to be interpreted as described in BCP 14 (RFC 2119 and RFC 8174) when, and only when, they appear in all capitals.
